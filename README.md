# Blank

Коллекция настольных игр.

## Игры

### Крестики-нолики

Классическая игра для двух игроков на поле 3×3.

#### Возможности:

- Два игрока играют за X и O
- X ходит первым
- «Призрачный» символ при наведении мышки на свободную клетку
- Подсветка последнего хода противника
- Плавные анимации появления символов
- Автоматическое определение победы и ничьи
- Возможность начать новую игру

#### Как играть:

1. Откройте `index.html` в браузере
2. Игроки ходят по очереди, кликая на свободные клетки
3. Цель — выстроить три своих символа в ряд (по горизонтали, вертикали или диагонали)
4. Игра определит победителя или объявит ничью
5. Нажмите «Новая игра» для начала новой партии
