# Коллекция игр / Games Collection

Коллекция пошаговых настольных и интеллектуальных игр на основе телевизионных передач, написанных на C#.

## О проекте

Это проект по созданию коллекции различных игр. Каждая игра создаётся как отдельное простое приложение, которое можно запустить и поиграть.

## Текущие игры

### 1. Крестики-нолики (Tic-Tac-Toe)

Классическая игра в крестики-нолики для двух игроков.

**Особенности:**
- Консольный интерфейс
- Игра для двух игроков за одним компьютером
- Проверка победных комбинаций
- Определение ничьи
- Возможность играть повторно

**Запуск:**
```bash
dotnet run --project Games/TicTacToe/TicTacToe.csproj
```

## Технологии

- **C#** - основной язык разработки
- **.NET 8.0** - целевая платформа
- **Консоль** - интерфейс для простых игр

## Структура проекта

```
GamesCollection/

│   └── TicTacToe/          # Крестики-нолики
```

## Добавление новой игры

Каждая игра создаётся как отдельный C# проект в папке `Games/`:

1. Создайте папку для игры: `Games/ИмяИгры/`
2. Создайте файл проекта `ИмяИгры.csproj`
3. Реализуйте игровую логику
4. Обновите README с описанием игры
