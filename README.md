# Коллекция игр / Games Collection

Большая коллекция пошаговых настольных и интеллектуальных игр на основе телевизионных передач с поддержкой мультиплеера.

## Описание проекта

Проект представляет собой систему для создания и проведения многопользовательских игровых сессий. Включает в себя хаб-приложение (лаунчер) и набор игр с единой архитектурой.

### Основные возможности

- **Хаб-лаунчер** в стиле Microsoft Office для управления играми
- **Мультиплеер** - игра по сети с другими участниками
- **Система ролей** - игроки, модераторы, организаторы, ведущие и др.
- **Игровые комнаты** с чатом и управлением участниками
- **Гибкая архитектура** для добавления новых игр

### Типы ролей

- **Игрок** - обычный участник игры
- **Модератор** - управление чатом и порядком
- **Организатор** - создатель комнаты с полными правами
- **Ведущий** - для телевизионных игр
- **Оператор** - технический специалист
- **Редактор** - управление контентом
- **Наблюдатель** - просмотр без участия

## Технологии

- **C#** - основной язык разработки
- **WPF** - графический интерфейс (текущая версия)
- **Unity** - планируется для более сложных игр
- **ASP.NET** - планируется для серверной части мультиплеера
- **.NET 8.0** - целевая платформа

## Быстрый старт

### Требования

- Visual Studio 2022 или новее
- .NET 8.0 SDK
- Windows 10/11

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/NikitaGamer64/Blank.git
cd Blank

# Сборка проекта
dotnet build

# Запуск хаба
dotnet run --project src/GamesHub/GamesHub.csproj
```

```csharp
using GamesCore.Models;

public class MyGame : BaseGame
{
    public MyGame() : base("Моя игра", "Описание", 2, 4) { }

    public override void Initialize() { /* логика */ }
    public override void Start() { /* логика */ }
    public override void Pause() { /* логика */ }
    public override void End() { /* логика */ }
}
```

### Создание игровой комнаты

```csharp
var game = new MyGame();
var room = new GameRoom("Моя комната", game, maxPlayers: 4);
room.AddPlayer(new Player("Игрок 1"));
```
